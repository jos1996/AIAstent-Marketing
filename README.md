# Workly AI Marketing Website

Modern marketing website for Workly AI - Your Real-Time AI Interview Copilot & Work Assistant.

## Features

- AI Interview Assistant & Interview Copilot
- Intelligent Screen Analysis
- Smart Reminder Setup
- Conversation Rewriter
- Invisible While Sharing Mode
- Real-time Code Solutions

## Tech Stack

- React 18
- TypeScript
- Vite
- CSS-in-JS (inline styles)

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This site can be deployed to:
- Netlify
- Vercel
- GitHub Pages
- Any static hosting service

